Rails实战之B2C商城开发 系列大课程(不断更新中)

Rails实战之B2C商城开发的大课程来了，在这个课程中我们会从实际需求的角度出发，给大家讲解电商平台核心功能的设计以及开发实战，不会涉及Ruby和Rails基础理论的讲解，这个课程中将会都是满满的干货

我们的目标是在这个大课程中，从需求出发，让你掌握电商网站的核心功能是如何设计和开发的

目前计划中的功能点有:

  • 用户模块: 用户注册登录 / 手机验证码 / 忘记密码 / 修改密码 ／ 角色管理
  • 商品模块: 分类 / 商品浏览 / 搜索 / 商品标签
  • 购物车模块: 购物车 / 订单
  • 支付模块: 支付宝 / 微信支付 / PayPal
  • 后台管理模块: 用户 / 商品 / 订单 / 付款记录
  • 营销模块: 邮件 / 促销活动
  • 运维模块: 系统配置 / 异步任务
  • 权限管理模块
  • 等等

学习该课程需要你有一定的Ruby和Rails开发基础，以及前端HTML, CSS, JavaScript的使用经验。

同样，如果你有想要了解的相关内容，可以联系我们。
